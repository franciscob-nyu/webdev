About

This webapp was developed by Francisco Briggs as a final project for Wed Dev class at NYU. It is intended as a parody of internet mysteries, as well as being inspired by the low-fi aesthetics of Yume Nikki and Undertale.

Usage

This webapp is controlled entirely by mouse movement and clicks. You can check it out here!

They say of you go through enough of those mind-bending portals, a mysterious character will appear, the one behind this madness. Or at least, that's what the rumors say.

Description

This webapp uses the GameMakerStudio1 development environment, which provides an interface to handle more complex interactive projects. The entire webapp is rendered inside a JS canvas. You can view a screenshot of the development environment with some of my code here. I can also provide the entire source code if needed. However, it is optimally viewed with the GMS development environment.
